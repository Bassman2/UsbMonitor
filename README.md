# UsbMonitor
Common repository for WpfUsbMonitor and FormsUsbMonitor

# WpfUsbMonitor
WPF USB Monitor to detect if a USB device is plugged in or out

# FormsUsbMonitor
Forms USB Monitor to detect if a USB device is plugged in or out
